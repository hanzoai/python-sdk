## HIP Process

1. **Draft**: Initial proposal with design rationale
2. **Review**: Community feedback and iteration
3. **Accepted**: Approved for implementation
4. **Final**: Fully implemented and documented

## Contributing

To propose a new HIP:

1. Create a new markdown file: `HIP-XXXX.md`
2. Follow the template structure (Abstract, Motivation, Design, Implementation)
3. Submit a PR for review
